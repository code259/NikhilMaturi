# Nikhil Maturi Personal Website

A Vercel-ready Next.js personal site for research, writing, and contact info.

## Run locally

```bash
npm install
npm run dev
```

## Content

- Research projects live in `src/lib/data.ts`.
- Blog posts are Markdown files in `content/blog`.
- Interactive blog blocks are registered in React components, starting with the graph-message-passing placeholder.

## Verify

```bash
npm run typecheck
npm run build
```
